### Lonely Warrior

## Controls
**WASD** - Movement

**SPACE** - Start game, shoot

**R** - Reload

**1, 2, 3, 4** - Pistol, Machine Gun, Shotgun, Rocket Launcher

## Cheat Codes
**M + P, F, S, R** - Infinite ammo for Pistol, Machine Gun, Shotgun and Rocket Launcher accordingly
